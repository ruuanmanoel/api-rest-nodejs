# RF

- [ ] O usuário deve poder uma nova transação;
- [ ] O usuário deve poder obter resumo da sua conta;
- [ ] O usuário deve poder listar todas as transações que já ocorreram;
- [ ] O usuário deve poder visualizar uma única transação;

# RNF

- [ ] A transação pode ser do tipo crédito que somará ao valor total, ou débito que irá subtrair;
- [ ] Deve ser possível identificar o usuário entre as requisições;
- [ ] O usuário so pode visualizar transações o qual ele criou;
